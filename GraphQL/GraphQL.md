## GraphQL
- 페이스북에서 만든 쿼리 언어
- 서버사이드에 들어온 gql로 작성된 커리를 받아 처리한 결과를 돌려줌
    - sql과 같은 쿼리 언어이지만, 구조적 차이는 매우 큼

## 구조

#### 쿼리/뮤테이션

```GraphQL
{
    hero {
        name
    }
}
```

- 상당히 직관적인 쿼리
- 쿼리는 R에 뮤테이션은 CUD에 사용
